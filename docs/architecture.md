# `utils.io`

### `InputParser`
* **! ОТВЕЧАЕТ ТОЛЬКО ЗА ПЕРВОНАЧАЛЬНУЮ ОБРАБОТКУ ВВОДА !**
* Ждет ввода команды (в отдельном потоке, наверное)
* Парсит команду и ее аргументы
* Валидирует ввод и готовит сообщение об ошибке
* Получает `Executable` с помощью `CommandFabric`
* ??? Добавляет команду `Executable` в очередь выполнения, если она затрагивает игровой процесс, или выполняет команду сам

### `CommandFabric`
* По константе `InputCommand` создает экземпляр команды `Executable` с аргументами

### `Executable`
* Интерфейс команд
* Имеет метод `void execute(String[] args)`

### `Command`
* Абстрактный класс команд
* Реализует `Executable`
* Имеет метод `Object[] convertArgs(String[] args)`

### Реализации `Command`
* ??? Используют в работе `execute()` методы `GameManager`, если команды касаются игрового процесса

### `InputHandler`
