## Инструкция по запуску

---

В репозитории уже есть готовый артефакт (JAR-архив). Чтобы его запустить, в консоли необходимо ввести одну из следующих команд в зависимости от требуемого режима работы:
```
# Консольный режим (CLI)
java -jar ivk-app.jar --cli
```
```
# Веб-режим
java -jar ivk-app.jar --web
```
После запуска доступно по адресу:
```
http://localhost:8080/api/
```

Для запуска необходимо иметь установленной Java 8+.

## Краткое описание

---

Структура приложения описана в [документации](docs/architecture.md).

Описание задания доступно по [внешней ссылке](https://ivk.ru/developer/testovoe-zadanie_igra-kvadraty/).

Фронтенд-часть приложения доступна по [ссылке](https://github.com/ahmadeev/ivk-frontend).

Компромисы:
* Разработка CLI-версии в ветке `master` представлена не с самого начала. Первые наброски программы остались в ветке `main-old`.
* Spring Boot для веб-приложения был выбран ввиду наличия embedded-сервера. Это упростило процесс запуска.
* В веб-версии представлена другая сущность ответа (`GameResponseEntity` вместо `SimpleMoveDTO`) для сохранения возможности получения хода и результатов проверки за один HTTP-запрос.
