### Инструкция по запуску
В репозитории уже есть готовый артефакт (JAR-архив). Чтобы его запустить, в консоли необходимо ввести одну из следующих команд в зависимости от требуемого режима работы:
```
# Консольный режим (CLI)
java -jar ivk-app.jar --cli
```
```
# Веб-режим
java -jar ivk-app.jar --web
```

Структура приложения описана в [документации](docs/architecture.md).
